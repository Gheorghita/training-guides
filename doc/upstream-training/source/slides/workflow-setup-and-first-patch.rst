Exercise
========

- Configure your workspace
    - ``git config``
    - install git review
    - ``git clone``
    - ``git review -s``
- Create & push a Work in Progress (WIP) to the sandbox
    - ``git checkout -b <branch_name>``
    - Make changes (add / edit files)
    - Commit files and write commit message
    - ``git review``
